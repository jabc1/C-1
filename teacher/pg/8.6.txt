配置YUM源：
1、把软件包放到某一目录下
cp /mnt/Server /var/ftp/pub -r

2、生成包的依赖关系文件
[root@server Server]# rpm -ivh createrepo-0.4.11-3.el5.noarch.rpm 
[root@server Server]# createrepo /var/ftp/pub/Server/


配置Yum工具，指定Yum源的位置 
vim /etc/yum.conf

 19 name=redhat enterpise linux server 5.3 server
 20 #baseurl=file:///var/ftp/pub/Server
    baseurl=ftp://192.168.1.254/pub/Server

 21 enabled=1
 22 gpgcheck=1
 23 gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-redhat-release

总结：对于源代码包
有绿色包，解压就可用

非绿色包，要解压再安装

1、解压
tar zxvf xxxx.tar.gz
tar jxvf xxxx.tar.bz2
2、cd到解压生成的目录
3、找readme和install文件。注意大小写的文件名字都有可能。一般readme文件中有包的功能描述。而install文件中会有包的安装方法描述。

通用 的源代码包的安装方法：
以安装第三方的NTFS-3G程序，来让当前系统支持对NTFS分区的读写。已经下载好了源代码包。

1 、解压源代码 包，并进入到相应目录中，找INSTALL文件和README文件。
]# tar zxvf ntfs-3g-2009.4.4.tgz 
]# cd ntfs-3g-2009.4.4
]# ls INSTALL README 
INSTALL  README
2、根据INSTALL文件中的提示进行编译安装
./configure --prefix=/usr/local/ntfs-3g
make 
make install

]# mount -t ntfs-3g /dev/sda1 /net
FATAL: Module fuse not found.
ntfs-3g-mount: fuse device is missing, try 'modprobe fuse' as root

安装fuse模块(rpm -q kernel-devel )
tar zxvf fuse.....tar.gz
cd fuse...
./configure --prefix=/usr/local/fuse
make
make install

加载fuse模块到内核：
modprobe  fuse
insmod /lib/modules/2.6.18-128.el5/kernel/fs/fuse/fuse.ko
